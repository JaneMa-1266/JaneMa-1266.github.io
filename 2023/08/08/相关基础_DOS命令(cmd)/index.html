DOS & cmd

发表于2023-08-08|更新于2023-08-13|编程
|字数总计:1.2k|阅读时长:4分钟|阅读量:

DOS（disk operating system）磁盘操作系统，
是一个顶层系统，在Windows GUI（图形界面）出来以前就是人和机器交互的工具，
当时人和机器交互就只能通过命令而不能使用鼠标，这种情形有点类似我们在电脑蓝屏重启时按F8进入安全模式。
在Windows GUI问世后，DOS事实上已不再需要，但其功能却以程序的形式被保留下来，这个程序在Windows中就是cmd.exe。

cmd.exe windows下的命令提示符，

​ DOS是一种纯命令方式，cmd其实就是在windows状态下进入dos方式
​ cmd.exe使用的命令行和DOS的命令大部分是一致的，所以可以认为cmd.exe和DOS有继承关系，但cmd是一个程序，DOS是一个操作系统，只是恰巧命令一样而已。（cmd 在windows下是DOS模拟器,能运行绝大部分DOS命令,但是不是真正的DOS；DOS是一个操作系统,而CMD只是在WINDOWS下面模拟DOS环境）
————————————————
在DOS命令窗口中编写DOS命令-win+R-cmd

常见DOS/cmd命令

启动cmd
  1. 用户启动，Win+r 输入cmd，Enter
  2. 管理员启动，Win+r 输入cmd，Ctrl+Shift+Enter
文件命令

exit - 退出当前DOS命令窗口

cls - clear screen/清屏

复制 - 选中要复制的内容，在任意位置点击鼠标右键

ctrl+c - 终止命令

Tab按键 - 补全命令

dir - directory/列出当前目录下所有的子文件夹/子目录（显示详细信息） 【同linux的”ls”命令】

cd - change directory/改变目录 - cd+目录的路径(可以是相对路径or绝对路径)

​ cd .. 回到上级目录；cd \ 直接回到根目录 （cd ../../../ 回到上三级目录）

​ 路径中..代表上级目录；.代表当前目录

C：或D： - 直接切换盘符

md - 创建目录

rd - 删除目录

网络相关

ipconfig - 查看IP地址

ping - 测试网络连通性

netstat - 查看网络连接状态
